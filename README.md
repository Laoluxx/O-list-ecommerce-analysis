# Olist SQL E-commerce Analysis

## Project Overview
This project is part of my Data Analytics Portfolio and focuses on analyzing Brazilian e-commerce data from the **Olist dataset** using **MySQL**.

The goal is to answer important business questions around:

- order trends
- revenue performance
- product performance
- seller performance
- delivery efficiency
- customer behavior

This project demonstrates my ability to:

- design and create relational databases in MySQL
- import and validate large CSV datasets
- write clean, portfolio-ready SQL queries
- perform business-focused exploratory analysis
- organize SQL projects using a professional GitHub structure

## Analysis Questions 
This project answers key e-commerce business questions, including:

How does order volume change over time?

How does revenue trend monthly?

Which product categories sell the most?

Which sellers generate the most revenue?

What is the average delivery time?

What is the distribution of order statuses?

Who are the top customers by total spend?

Which product categories receive the best reviews?

What percentage of deliveries are late?

What is the repeat purchase rate?

Which products generate the highest revenue?

Which product categories generate the most revenue?

What is customer lifetime value (CLV)?

Which cities place the most orders?

Key SQL Analyses
1. Monthly Order Volume Trend

Tracks total orders by month to identify seasonality and sales patterns.

2. Monthly Revenue Trend

Measures revenue growth over time.

3. Top Product Categories by Orders

Highlights the most frequently purchased product categories.

4. Top Sellers by Revenue

Identifies top-performing sellers by revenue.

5. Average Delivery Time

Measures logistics and delivery performance.

6. Orders by Status

Shows the distribution of delivered, shipped, canceled, and other order statuses.

7. Top Customers by Revenue

Identifies customers generating the highest total spend.

8. Average Review Score by Product Category

Compares customer satisfaction across product categories.

9. Late Delivery Rate

Measures the percentage of orders delivered later than expected.

10. Customer Repeat Purchase Rate

Evaluates customer retention and repeat buying behavior.

11. Top Products by Revenue

Shows products contributing the most revenue.

12. Revenue by Product Category

Ranks product categories by revenue contribution.

13. Customer Lifetime Value (CLV)

Calculates total revenue generated per customer.

14. Top Cities by Order Volume

Identifies cities with the highest order activity.
